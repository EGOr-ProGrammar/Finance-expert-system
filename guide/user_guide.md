# Руководство пользователя (User Guide)

Добро пожаловать в систему поддержки принятия решений на основе нечеткой логики.

## 1. Начало работы
Для получения прогноза или рекомендации запустите модуль пользователя:
```powershell
./FuzzyExpertSystem_User.exe
```

## 2. Процесс консультации

### Шаг 1: Загрузка базы знаний
При запуске программа автоматически считывает файл `data/knowledge_base.json`. Если файл поврежден или отсутствует, программа сообщит об ошибке.

### Шаг 2: Ввод фактов
Программа по очереди запросит у вас значения для всех параметров, которые учитывает система. 
*   *Пример:* `Enter value for [Profitability]:`.
*   Вводите численные значения (целые или с плавающей точкой).

### Шаг 3: Интерпретация результата
После ввода всех данных система выдаст результат логического вывода.
*   Результат — это конкретное число (скаляр), полученное путем "взвешивания" всех сработавших правил.
*   *Пример:* `Output [Decision]: 7.5` может означать высокую степень уверенности в успехе (если шкала от 0 до 10).

## 3. Часто задаваемые вопросы

**В: Почему я получил "Warning: No rules were activated"?**
О: Это означает, что введенные вами данные являются экстремальными или не предусмотренными базой знаний. Попробуйте ввести другие данные или обратитесь к администратору для дополнения базы правил.

**В: Насколько точен результат?**
О: Система имитирует рассуждения эксперта. Точность зависит от того, насколько качественно эксперт настроил функции принадлежности и правила в экспертном модуле.

**В: Могу ли я изменить правила сам?**
О: Для изменения логики работы системы используйте `FuzzyExpertSystem_Expert.exe`.
